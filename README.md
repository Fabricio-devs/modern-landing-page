# Modern Landing Page

A modern front-end landing page showcasing responsive design, custom UI components, SVG illustrations, and smooth CSS animations built with HTML, CSS, and vanilla JavaScript.

## 🔗 Live Demo

👉 https://fabricio-devs.github.io/modern-landing-page/

## ✨ Features

- Fully responsive layout (desktop, tablet, mobile)
- Modern dark UI with gradient backgrounds
- Reusable and scalable UI components
- Smooth hover effects and CSS transitions
- Semantic HTML and clean structure
- Vanilla JavaScript (no frameworks)
- Optimized for performance and accessibility

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
  - Flexbox & Grid
  - Transitions & animations
- **JavaScript (ES6+)**
- **GitHub Pages** for deployment
